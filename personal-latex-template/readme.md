### Custom Commands

Use as the following:
```latex
\command_name{}{
    [...]
}
```

The list of the __pre-made__ commands:

```txt
\thm{}{}    - Theorem
\cor{}{}    - Corollary
\mlenma{}{} - Lemma
\mprop{}{}  - Proposition
\clm{}{}    - Claim
\wc{}{}     - Weak Claim
\thmcon{}{} - Theorem Conjecture
\ex{}{}     - Example
\dfn{}{}    - Definition
\dfnc{}{}   - Definition Conjecture
\qs{}{}     - Question
\pf{}{}     - Proof
\nt{}{}     - Note
```
